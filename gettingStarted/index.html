Getting Started

Install the package

$ npm install --save react-snackbar-alert

Add to your app

  • Import the SnackbarManager component
  • Import the CSS styles
  • Add the <SnackbarManager /> to your component and save it to a ref
  • Call create on the SnackbarManager ref to create notifications

Example usage

import React, { useRef } from 'react';

import { SnackbarManager } from 'react-snackbar-alert';
import 'react-snackbar-alert/styles/react-snackbar-alert.css';

export default function BaseExample() {
  const snackbarManager = useRef(null);

  function showSnackbar() {
    snackbarManager.current.create({
      message: 'Hello Snackbar!'
    });
  } 

  return (
    <div>
      <SnackbarManager ref={snackbarManager} />
      <main>
        <button onClick={showSnackbar}>Show Snackbar</button>
      </main>
    </div>
  );
}
